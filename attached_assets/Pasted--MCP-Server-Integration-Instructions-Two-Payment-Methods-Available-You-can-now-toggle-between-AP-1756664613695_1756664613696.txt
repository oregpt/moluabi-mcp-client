Two Payment Methods Available
You can now toggle between API Key billing and ATXP crypto payments by simply changing the endpoint URL:

Method 1: API Key (Account Billing)
const response = await fetch('https://moluabi-mcp-server.replit.app/mcp/call', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'list_agents',
    arguments: { apiKey: 'mab_your_api_key' }
  })
});
Method 2: ATXP (Crypto Payments)
// First, get tools list (no auth required)
const toolsResponse = await fetch('https://moluabi-mcp-server.replit.app/atxp', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    jsonrpc: '2.0',
    method: 'tools/list',
    id: 'get-tools'
  })
});
// Then, call tools with ATXP authentication
const response = await fetch('https://moluabi-mcp-server.replit.app/atxp', {
  method: 'POST',
  headers: { 
    'Content-Type': 'application/json',
    'Authorization': 'Bearer your_atxp_token'
  },
  body: JSON.stringify({
    jsonrpc: '2.0',
    method: 'tools/call',
    params: {
      name: 'list_agents',
      arguments: { apiKey: 'mab_your_api_key' }
    },
    id: 'call-tool'
  })
});
