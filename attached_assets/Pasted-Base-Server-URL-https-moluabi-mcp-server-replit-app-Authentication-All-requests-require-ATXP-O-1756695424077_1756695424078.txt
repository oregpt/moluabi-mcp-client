Base Server URL
https://moluabi-mcp-server.replit.app/
Authentication
All requests require ATXP OAuth token in the Authorization header:

Authorization: Bearer YOUR_ATXP_TOKEN
MCP Request Format
All requests use JSON-RPC 2.0 format with POST to the root endpoint.

1. List Available Tools ($0 - no payment required)
curl -X POST "https://moluabi-mcp-server.replit.app/" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ATXP_TOKEN" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/list",
    "params": {},
    "id": 1
  }'
2. Create Agent ($0.05 USDC)

      "name": "create_agent",

Expected Response Format
All successful responses return:

{
  "jsonrpc": "2.0",
  "result": {
    "content": [
      {
        "type": "text", 
        "text": "Success message with payment confirmation and data"
      }
    ]
  },
  "id": REQUEST_ID
}
Authentication Errors
Without valid ATXP token:

{
  "error": "invalid_token",
  "error_description": "Token is not active"
}
Payment Required
Each tool call (except tools/list) requires USDC payment through ATXP before execution. The payment is processed automatically when the tool is called with a valid token.
